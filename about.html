About HorizonHub

Smart travel booking platform built by students, for travelers

The Problem We Solved

Tourists visiting destinations like Mustang and Pokhara often face confusion while booking trips online. Pricing information is frequently vague or hidden, travel websites are overly complex, and calculating the total cost per person is difficult. These issues create frustration and uncertainty for travelers.

HorizonHub directly addresses these problems by providing a transparent, straightforward, and interactive travel booking experience where users can see exactly what they're paying for.

Our Mission

To create a modern and responsive travel website that provides transparent pricing, allows real-time trip cost calculation, and improves user experience through UI/UX principles. We aim to simplify travel planning for domestic and international tourists, students, families, and budget-conscious travelers.

Key Features

🎴 Interactive Trip Cards

Attractive destination cards with high-quality images, descriptions, and ratings.

🔍 Live Search & Filter

Instantly search and filter destinations by name, budget, or category without page reload.

💵 Per-Person Cost Calculator

Real-time price calculation based on travelers, hotel tier, and transport options.

📝 Simulated Booking System

Complete booking form with localStorage to save your trip information locally.

📱 Responsive Design

Seamless experience across smartphones, tablets, and desktop computers.

🎨 Modern UI/UX

Clean design following principles of simplicity, consistency, and accessibility.

Technology Stack

HTML5

Semantic structure and baseline layout

CSS3

Flexbox, Grid, animations, responsive design

JavaScript

Dynamic search, filtering, cost calculation

localStorage

Client-side data persistence
